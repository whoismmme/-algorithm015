**Q: 写一个关于 HashMap 的小总结。**

HashMap采用数组+链表的结构实现存储
存储：put 的时候是先计算 hash 数组的index，判断当前位置是否有条目，无冲突则直接放入，如果有冲突，则通过链表或者红黑树解决冲突。
读取：get 和 put 基本类似，判断 hash table 是否为空，不是则根据 put 理论来找到对应数值是否存在
